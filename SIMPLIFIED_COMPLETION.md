## 📌 项目概览

为国际贸易团队打造的**极致简洁运费计算工具**。用户只需：
1. 选择目的国家

无需复杂操作，一屏显示所有信息。

## 🎯 核心成果

### ✨ 简化版设计
```
原始版本 → 新简化版本

原始：多个输入框（国家、重量、尺寸、季节费、燃油费）
      两栏布局（表单 + 结果）
      复杂交互流程

新版：只有2个输入框（国家、重量）
      单卡片布局
      实时计算，无需交互
```

| **交互** | 实时计算，无需按钮，一键复制 |
| **信息** | 基础运费 + 燃油费 + 总费用 + 提示 |

### 📱 响应式支持

```typescript

                                        ↓
<h1 className="text-2xl font-bold">你的标题</h1>
```

### 修改配色
顶部条颜色 (第23行)：
```typescript
<div className="bg-gradient-to-r from-blue-600 to-blue-700">
                                    ↓
<div className="bg-gradient-to-r from-indigo-600 to-purple-700">
```

### 修改燃油费率

```typescript
rate: 0.18,  // ← 改成 0.20 或 0.15
```
