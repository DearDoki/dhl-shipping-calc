### 场景2：美国 25kg 邮件

```
输入: 美国, 25kg

计算过程：
1. 美国 → 分区6
2. 25kg × 分区6价格 = ¥1,800.00 (基础运费)
3. 燃油费 = ¥1,800.00 × 18% = ¥324.00
4. 总费用 = ¥1,800.00 + ¥324.00 = ¥2,124.00
```

## 🔢 数据流向

```
用户输入
   ↓
┌─────────────────────────┐
│ 国家查询 (countries.ts)  │ ← 100+国家数据
│ name → partition, remote │
└─────────────────────────┘
   ↓
┌─────────────────────────┐
│ 价格查询 (prices.ts)     │ ← 9个分区价格表
│ partition, weight →     │
│ baseCost                │
└─────────────────────────┘
   ↓
┌─────────────────────────┐
│ 计算引擎 (calculator.ts) │ ← 燃油费配置
│ baseCost × (1 + 0.18)   │
│ = totalCost             │
└─────────────────────────┘
   ↓
显示结果 (UI组件)
```

### 示例1：香港 → 美国 8kg

| 项目 | 数值 |
|------|------|
| 国家 | 美国 |

## 🎓 代码示例

### 如何调用计算函数

```typescript
import { calculateShipping } from './data/calculator'

const result = calculateShipping(
  weight: 5.5,        // kg
  partition: 7,       // 分区号
  fuelRate: 0.18,     // 18%
  applySeasonal: false // 是否应用季节费
)

console.log(result)
// {
//   baseCost: 658.50,
//   fuelSurcharge: 118.53,
//   seasonalSurcharge: 0,
//   totalCost: 777.03
// }
```

### 如何查询国家信息

```typescript
import { findCountryByName, getCountriesByPartition } from './data/countries'

// 按名称查询
const country = findCountryByName('英国')
// { name: '英国', partition: 7, isRemote: false, code: 'GB' }

// 按分区查询
const partition7Countries = getCountriesByPartition(7)
// [{ name: '英国', ... }, { name: '法国', ... }, ...]
```
