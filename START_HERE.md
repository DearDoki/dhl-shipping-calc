# ⚡ 3分钟快速启动

## 📦 准备工作

1. **打开PowerShell或终端**
2. **切换到项目目录**：
   ```powershell
   cd c:\Users\Administrator\WorkBuddy\20260331142200\dhl-shipping-calc
   ```

## 🚀 启动步骤

### 第1步：安装依赖（首次需要）
```powershell
npm install
```
⏱️ 耗时：1-2分钟

### 第2步：启动开发服务器
```powershell
npm run dev
```
✅ 出现类似信息：
```
VITE v5.0.0  ready in 234 ms

➜  Local:   http://localhost:5173/
➜  press h to show help
```

### 第3步：打开浏览器
点击链接或在浏览器输入：
```
http://localhost:5173
```

## 🛑 遇到问题？

| 问题 | 解决方案 |
|------|--------|
| 端口被占用 | 改用其他端口：`npm run dev -- --port 5174` |
| npm 不识别 | 检查Node.js是否安装：`node --version` |
| 页面加载缓慢 | 清理缓存：`rm -rf node_modules/.vite` |
| 计算不对 | 检查燃油费率是否最新 |
