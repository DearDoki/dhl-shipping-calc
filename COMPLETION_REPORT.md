## 📊 项目概览

### 什么是这个项目？

一个**全栈Web应用**，用于DHL Express国际快递运费的快速自动计算。
- 🌍 支持**100+国家**
- 💰 **完整价格表**（0.5kg-99,999kg）
- ⚡ **实时燃油费计算**
- 📱 **响应式设计**
- 🎨 **现代UI界面**

### 核心特性

✅ **一键计算** - 输入国家和重量，秒速得出运费  
✅ **精准计价** - 支持0.5kg向上取整，体积重量计算  
✅ **动态费率** - 实时燃油费、可选冬季费  
✅ **智能提示** - 偏远地区警告、详细费用明细  
✅ **即开即用** - 无需配置，开箱即用

## 🎯 计费规则速览

```
基础运费 = 查询价格表(分区, 重量)
           ↓
燃油费 = 基础运费 × 18%
           ↓
季节费(可选) = 重量 × 3.5元/kg
           ↓
━━━━━━━━━━━━━━━━━━━━
总运费 = 基础运费 + 燃油费 + 季节费
```

**重量规则:**
- 最小单位：0.5kg
- 取整方向：向上取整
- 体积重量：长×宽×高÷5000
- 计费采用：MAX(实际重量, 体积重量)
