# 🎬 5分钟快速开始 - 运行爬虫

> 如果你着急，就看这一页！其他详细内容在 `爬虫新手教程.md` 中。

---

## 第1步：打开 PowerShell（命令行窗口）

**操作**：
1. 按下键盘上的 `Win` 键（有Windows标志的）
2. 同时按住 `R` 键
3. 会弹出一个小窗口，输入：
   ```
   powershell
   ```
4. 按 `Enter`

**你会看到**：一个蓝色的窗口打开，就像这样：
```
Windows PowerShell
版权所有 (C) Microsoft Corporation。
...
PS C:\Users\Administrator>
```

✅ **完成第1步！**

---

## 第2步：进入项目文件夹

**操作**：在蓝色窗口中粘贴这个命令：
```powershell
cd c:\Users\Administrator\WorkBuddy\20260331142200\dhl-shipping-calc
```

然后按 `Enter`

**你会看到**：
```
PS C:\Users\Administrator\WorkBuddy\20260331142200\dhl-shipping-calc>
```

✅ **完成第2步！**

---

## 第3步：检查是否安装了必要的软件

### 检查 Node.js

**操作**：输入
```powershell
node -v
```
按 `Enter`

**你会看到**：
- ✅ `v20.10.0` 或类似 → **已安装，继续！**
- ❌ `'node' 不是内部或外部命令` → 需要安装（见下方）

### 检查 Python

**操作**：输入
```powershell
python --version
```
按 `Enter`

**你会看到**：
- ✅ `Python 3.11.0` 或类似 → **已安装，继续！**
- ❌ `'python' 不是内部或外部命令` → 需要安装（见下方）

**如果需要安装软件**：
- Node.js: https://nodejs.org/ (下载 LTS 绿色按钮)
- Python: https://www.python.org/downloads/

✅ **完成第3步！**

---

## 第4步：安装项目依赖（第一次做）

**操作**：在 PowerShell 中输入
```powershell
npm install
```
按 `Enter`

**你会看到**：
- 一堆文字滚过来（在下载东西）
- 最后显示 `added 500+ packages` 之类的

⏳ **这需要 2-5 分钟，等着就行。不要关闭窗口！**

✅ **完成第4步！**

---

## 第5步：安装爬虫依赖库（第一次做）

**操作**：输入
```powershell
pip install playwright requests beautifulsoup4
```
按 `Enter`

⏳ **等待 1-3 分钟...**

**然后继续输入**：
```powershell
playwright install chromium
```
按 `Enter`

⏳ **这个会花 3-5 分钟，下载一个虚拟浏览器（200MB）**

当你看到 `To verify Playwright is installed correctly` 这样的信息时 → ✅ **成功了！**

---

## 🚀 第6步：运行爬虫！

**操作**：输入
```powershell
python scripts/fuel_surcharge_scraper.py
```
按 `Enter`
