## 什么是爬虫？

**爬虫就像一个"机器人助理"**，它可以：
- 🤖 自动打开网页（就像你用浏览器一样）
- 👀 自动阅读网页内容
- 📋 自动提取你需要的信息（比如燃油费百分比）
- 💾 自动保存到你的项目文件中

**在这个项目中**，爬虫会：
1. 自动打开 DHL官网燃油费页面
2. 自动找到燃油费百分比（比如 18%）
3. 自动更新你的计算器

## 准备工作

### ✅ 检查你的电脑上是否有这些软件

#### 1️⃣ 检查是否装了 Node.js

**步骤**：
1. 按 `Win + R` 键，会弹出一个小窗口
2. 输入 `powershell` 然后按 `Enter`（会打开一个蓝色的命令行窗口）
3. 在窗口中输入：
   ```
   node -v
   ```
4. 按 `Enter`

**结果**：
- ✅ 如果显示 `v20.x.x` 或类似版本号 → **已安装** ✓
- ❌ 如果显示 `'node' 不是内部或外部命令...` → **需要安装**

**如果需要安装**：
- 访问 https://nodejs.org/
- 下载 **LTS 版本**（绿色的按钮）
- 一直点"下一步"安装

   按 `Enter`

3. **运行爬虫**
   ```powershell
   python scripts/fuel_surcharge_scraper.py
   ```
   按 `Enter`

4. **看到结果**
   
   爬虫会自动：
   - 打开一个浏览器窗口（可能是虚拟的，你看不到，但确实在运行）
   - 访问 DHL 网站
   - 提取燃油费百分比

   当你看到类似这样的信息：

## 常见问题解答

### Q1: 我看不到爬虫在做什么？

**A**: 这是正常的！因为爬虫使用的是"虚拟浏览器"（headless）。你看不到它打开网页，但它确实在后台工作。

如果想看到浏览器窗口，可以编辑脚本，改成：
```python
browser = await p.chromium.launch(headless=False)  # headless=False 就能看到了
```

---

### Q2: 爬虫运行了，但说找不到燃油费？

**A**: 可能是以下原因：

1. **网站改版了** → DHL 官网可能更新了页面结构
   - 解决：告诉我，我会更新爬虫代码

2. **网络连接问题** → 爬虫无法访问网站
   - 解决：检查你的网络是否正常

3. **依赖没装全** → 少装了某个库
   - 解决：重新运行 `pip install playwright requests beautifulsoup4`

---

### Q3: 爬虫会损坏我的计算器吗？

**A**: **不会！** 爬虫只会：
- ✅ 更新燃油费百分比
- ✅ 不会删除其他数据
- ✅ 每次都会备份旧的配置

如果出问题，你可以随时手动改回来。

---

### Q4: 我能手动修改燃油费吗？

**A**: 当然可以！

**文件位置**：
```
src/data/calculator.ts
```

**找这一行**：
```typescript
export const DEFAULT_FUEL_SURCHARGE = 0.18;  // 18%
```

**改成你想要的数字**：
- `0.18` = 18%
- `0.20` = 20%
- `0.15` = 15%

改完后，项目会自动刷新计算器。

---

### Q5: 怎样停止自动运行？

**A**: 如果你不想让爬虫自动运行，可以：

1. 进入 WorkBuddy 的自动化管理界面
2. 找到 "DHL燃油费自动更新爬虫"
3. 改成 "暂停" 状态

这样就不会自动运行了。

---

### Q6: 爬虫失败了怎么办？

**A**: 别慌！按这个步骤排查：

1. **查看错误信息**
   ```powershell
   python scripts/fuel_surcharge_scraper.py
   ```
   会显示具体的错误

2. **最常见的错误**：
   - `Module not found` → 某个库没装，重新运行 `pip install playwright requests beautifulsoup4`
   - `Timeout` → 网络太慢或网站无法访问，等一下再试
   - `Element not found` → 网站改版了，告诉我更新代码

3. **看看日志文件**
   ```
   logs/scraper_log.txt
   ```
   里面会详细记录发生了什么

---
