# 🚀 DHL 运费计算器 - 简化版

## 📱 什么是简化版？

这是一个**极致简洁的网页版本**，用户只需要：

1. **选择国家** - 从下拉框选择目的地
2. **输入重量** - 输入邮件重量（kg）
3. **一键出价** - 系统即时计算运费

**特点**：
- ✨ 单页卡片设计，一屏显示全部信息
- ⚡ 实时计算，无需点击按钮
- 📱 移动友好，竖屏也能快速查询
- 💼 商务风格，深蓝配色
- 🎯 直接高效，0 干扰因素

## 💾 生产部署

### 构建优化版本
```bash
npm run build
```

生成的 `dist/` 文件夹包含可直接部署的静态文件。

### 部署选项

#### 选项1：腾讯CloudBase (推荐)
```bash
npm install -g @cloudbase/cli
cloudbase init
cloudbase deploy
```

#### 选项2：Vercel (最简单)
```bash
npm install -g vercel
vercel
```

#### 选项3：自建服务器
```bash
# 复制 dist/ 到服务器
scp -r dist/ user@server:/var/www/html/dhl-calculator/
```
